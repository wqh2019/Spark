# Spark Project

A lightweight Python agent framework with ReAct loop.

## Development

- Python 3.10+
- Uses `uv` for package management
- Run tests: `pytest`
- Lint: `ruff check`

## Code Style

- Line length: 100 characters
- Use type hints
- Docstrings for public APIs

## Environment

- `.env` contains real secrets (never commit)
- `.env.example` is the template (keep keys in sync with `.env`)

## Git

- Conventional commits: `feat:`, `fix:`, `docs:`, `chore:`, `test:`
- Push failures: retry up to 3 times
